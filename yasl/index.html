Yet Another Software License

Overview

YASL 2.0 is a modern and flexible software license created by YAS in March 2025, designed to provide maximum freedom while ensuring proper credit and responsible usage.

Key Features

  • Free use for any purpose
  • Commercial and personal usage permitted
  • Modification and redistribution allowed
  • Requires attribution to the original author
  • No warranties or liability included
  • Can be sublicensed under compatible terms
  • Allows inclusion in both open-source and proprietary projects
